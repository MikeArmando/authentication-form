A full-stack user management system built to learn Vanilla JavaScript, Node.js, and
relational databases.

## Key Features

- **Dynamic User Registration:** Capture user data via validated forms.
- **Modular Architecture:** Clean code structure using ES Modules.
- **Data Persistence:** Uses PostgreSQL as a database to keep data after refresh.
- **Live Validation:** Instantly validate users by their credentials.
- **Responsive Design:** Fully functional on mobile and desktop.
